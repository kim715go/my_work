# 전력 데이터 수집 및 분석 프로그램
온실가스센터에서 건물별 에너지 사용량 데이터 관리 업무를 맡으면서 직접 필요성을 느끼거나 사무실에서 필요성이 제기되어 개발한 프로그램들입니다.
윈도우 어플리케이션을 개발하기에 적합하지 않은 파이썬으로 제작되었고, 내부적으로만 사용할 계획으로 만들어진 프로그램이라 안정성 및 사용 편의성 측면에서 다소 불편한 점이 있습니다.
첫 두 프로그램은 학내 인트라넷 망에서만 작동하고 외부에서는 정상 작동하지 않아 직접 확인해 보시기는 어렵습니다. 또한 온실가스 알리미 샘플 프로그램은 건물별 전력 데이터가 반드시 로드되어야 작동하는데, 해당 데이터가 대외비라 공개된 웹 상에 게시하지 못했습니다.
대신 각 프로그램을 사용하는 장면을 녹화한 영상 링크를 첨부하였습니다.


## 오토베이스 캡처 프로그램
https://youtu.be/zM8Vzt6fT-8

센터에는 관내 건물별 전력 데이터를 조회할 수 있는 클라이언트 프로그램이 있습니다. 이를 '오토베이스'라고 부릅니다. 이 프로그램을 통해 학내 인트라넷 망을 통해 전력 데이터를 조회할 수 있습니다.
오토베이스는 데이터를 실시간으로 조회하는 것은 자유롭지만, 누적된 데이터를 다운로드받을 수 있는 기능은 막혀 있고, 서버에서 직접 다운로드받기 어렵다고 하여 데이터를 볼 수 있음에도 이를 가공할 수 있는 형태로 확보하지 못했습니다.
이 문제를 해결하기 위해 최초에는 먼저 오토베이스 화면을 자동으로 캡처한 후 이를 OCR로 읽어 데이터화하고자 하였습니다.
이 프로그램은 원하는 건물과 원하는 기간을 정해 주면 자동으로 돌면서 데이터 화면을 캡처하는 기능을 구현한 것입니다.

## 전력데이터 수집계산기
https://youtu.be/AjSUMGZaC8A

캡처 프로그램으로 화면 확보는 성공했으나, 업체로부터 제공받은 OCR 프로그램의 성능이 좋지 않고 이를 구동하는 데에도 손이 많이 가 사용하기가 쉽지 않았습니다.
이 문제를 해결하기 위해 주변 친구에게 의견을 물었고, 오토베이스가 http로 데이터를 주고받고 있으니 그걸 활용하여 데이터를 받아오면 될 것 같다는 조언을 받아 완전히 새로운 프로그램을 구상하였습니다.
수많은 계량기 중에 건물별 전력량을 계산하기에 필요한 계량기만을 골라 입력해 주면, 오토베이스에서 주고받는 패킷의 형태를 활용하여 데이터를 받아 정리하는 기능을 구현하였습니다.
또한, 수집된 데이터를 바탕으로 건물별 전력량을 계산하는 기능을 함께 구현하여 실제 작업에 유용하게 활용하고 있습니다.

## 데이터 체크 프로그램
https://youtu.be/s6ttk3NJbQs

전력데이터 수집계산기로 확보한 데이터는 이전 데이터에 비해 훨씬 더 고품질이지만, 여전히 여러 이유로 인한 오류가 존재했습니다.
오류 보정 프로그램이 있긴 했지만 오류 여부를 판단하는 로직이 너무 단순하게 되어 있어 사람이 직접 데이터를 1차적으로 검토할 필요성이 대두되었습니다.
이를 해결하기 위해 전력데이터 수집계산기에서 확보된 데이터의 내용을 쉽게 확인하고, 문제가 있을 것으로 의심되는 부분은 오류 보정 프로그램이 오류로 인식하는 0으로 쉽게 바꾸어 저장할 수 있도록 프로그램을 제작하였습니다.
이 프로그램은 데이터 의존성이 없기 때문에 직접 가동이 가능합니다. 게시한 A동 및 B동 csv 파일을 활용하여 직접 실험해보실 수 있습니다.

## 온실가스 알리미 샘플
https://youtu.be/F09TqxCIGA4

센터에서 진행한 여러 프로젝트 중, 각 기관별로 에너지 사용으로 인한 온실가스 배출량을 손쉽게 확인할 수 있는 프로그램을 제작, 배포하는 프로젝트의 일환으로 만든 프로그램입니다.
내부적으로 프리젠테이션에 활용하고, 실제 구현을 담당한 업체에게 프로토타입을 전달하고자 중요 기능을 위주로 구현한 샘플로, 프로그램을 구동할 때 반드시 온실가스 배출량 데이터가 로드되어야 하게 되어 있습니다.
다만, 기관별 및 건물별 에너지 사용 데이터는 게시할 수 없어 실제 구동은 되지 않습니다.
